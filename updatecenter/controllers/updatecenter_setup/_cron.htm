Automatic Updates:

  • You are running core module version | This feature requires core module version 1.13.0 or higher from github: https://github.com/damanic/ls1-module-core

Enabling Cron :To enable lemonstands cron feature your system scheduler must be set up to trigger the core cron process at an interval greater or equal to the interval value you enter below. Example command to add to your scheduler: /usr/local/bin/php -q /ROOT/PATH/TO/SITE/public_html/modules/core/cron.php

Note: Auto updates will not apply bleeding edge updates. Bleeding edge can only be done manually using force update
